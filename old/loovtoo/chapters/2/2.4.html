2.4 – CSS-i kasutamine

CSS (Cascading Style Sheets) võimaldab kujundada HTML-elemente. CSS-i saab lisada otse HTML-i sisse style atribuudi või <style> sildi abil, aga ka eraldi failina. CSS muudab lehe värve, fonte, paigutust ja palju muud.

<style>
h1 {
  color: #2563eb;
  text-align: center;
}
p {
  font-size: 1.2em;
}
</style>
<h1>Pealkiri</h1>
<p>See lõik on suurem ja pealkiri sinine.</p>

Pealkiri

See lõik on suurem ja pealkiri sinine.

  • style – otselemendi stiil
  • <style> – lehe päisesse lisatav CSS
  • Väline CSS – eraldi failina, mida lingitakse <link> sildiga

CSS-i õppimine võimaldab sul luua ilusaid ja professionaalseid veebilehti. Proovi muuta värve, fonte ja paigutust!

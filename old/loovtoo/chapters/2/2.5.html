2.5 – Stiilid ja klassid

HTML-is saab elementidele määrata klassid (class) ja id-d (id), et neid CSS-iga kujundada. Klassid võimaldavad sama stiili rakendada mitmele elemendile, id on unikaalne ühe elemendi jaoks.

<style>
.important {
  color: #e63946;
  font-weight: bold;
}
#highlight {
  background: yellow;
}
</style>
<p class="important">See tekst on tähtis.</p>
<p id="highlight">See tekst on esiletõstetud.</p>

See tekst on tähtis.

See tekst on esiletõstetud.

  • class – võimaldab määrata elemendile ühe või mitu klassi
  • id – unikaalne identifikaator ühe elemendi jaoks
  • CSS-is kasutatakse klassi ees . ja id ees #

Klasside ja id-de kasutamine muudab kujunduse paindlikuks ja koodi korrastatuks. Proovi ise erinevaid klasse ja id-sid kasutada!

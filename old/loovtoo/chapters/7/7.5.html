7.5 – Näidisfail allalaadimiseks

Laadi alla näidis-HTML-fail, mis sisaldab kõiki kursusel õpitud põhielemente. Vaata koodi, proovi seda muuta ja ava fail oma brauseris!

⬇ Laadi alla näidisfail

Näidisfaili kood

<!DOCTYPE html>
<html lang="et">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Näidis HTML-leht</title>
  <style>
    body { font-family: Arial, sans-serif; margin: 20px; }
    h1 { color: #2563eb; }
    table { border-collapse: collapse; margin: 10px 0; }
    th, td { border: 1px solid #ccc; padding: 4px 8px; }
    .important { color: #e63946; font-weight: bold; }
    #highlight { background: yellow; }
    .btn { background: #2563eb; color: #fff; border: none; padding: 6px 14px; border-radius: 5px; cursor: pointer; }
  </style>
</head>
<body>
  ... (vaata eelvaadet allpool) ...
</body>
</html>

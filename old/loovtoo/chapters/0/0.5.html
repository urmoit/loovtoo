0.5 – Esimese HTML-faili loomine

Kui oled valinud tekstiredaktori, saad alustada oma esimese HTML-faili loomist. Järgi neid samme:

  1. Ava tekstiredaktor (nt Notepad++ või VS Code).
  2. Loo uus fail.
  3. Kopeeri allaolev näide ja kleebi see faili.
  4. Salvesta fail laiendiga .html (nt minu_leht.html).
  5. Ava fail brauseris (tee failil topeltklõps või vali "Ava brauseris").
<!DOCTYPE html>
<html lang="et">
  <head>
    <meta charset="UTF-8">
    <title>Minu esimene leht</title>
  </head>
  <body>
    <h1>Tere, maailm!</h1>
    <p>See on minu esimene HTML-leht.</p>
  </body>
</html>

Tere, maailm!

See on minu esimene HTML-leht.

Nüüd oled loonud oma esimese HTML-lehe! Proovi muuta pealkirja või teksti, salvesta fail ja vaata muudatusi brauseris. Nii saad õppida HTML-i katsetades ja praktiseerides.
